# Generator UT SS
A simple generator for Yeoman.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator **locally**: `npm install -g generator-ut-ss`
- Run: `yo ut-ss`
